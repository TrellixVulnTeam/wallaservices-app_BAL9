# Hourglass
Hourglass és una plataforma d’intercanvi de serveis.
Permet als usuaris crear-se un perfil, pujar els serveis que ofereixen i xatejar amb altres membres per tal d’arribar a acords. 

La missió de la iniciativa és potenciar
l’economia col·laborativa entre la població.

Aquest Web realitzat per Ester Batllori i Dani Fernàndez és el projecte final del programa Singulars2020, una iniciativa de PIMEC que pretén formar a joves dins del sector del Desenvolupament Web.
